# Layer 0: Hardware

1. Prepare hardware, minimum of three nodes. For example, I am using Intel N150 mini PCs (GMKtec NucBox G3 Plus) for two of my nodes.
2. Hardware needs to be plugged in to a power source, preferably a UPS.
3. Hardware needs to be plugged in to an Ethernet cable, either an Ethernet switch or directly to a router.
4. During Layer 1 (Debian installation), a display should be plugged in (via HDMI), plus a USB keyboard (mouse is optional). These can be unplugged once installation is complete.
